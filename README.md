# washu-ai-navigator

## Project Structure

This repo is scaffolded as:

- `frontend/` - web UI (to be implemented)
- `backend/` - Python backend (FastAPI/other server can be added later)
- `data/` - data files (local cache, datasets, etc.)
- `prompts/` - prompt templates for agents

## Quick Start

### Backend

```bash
python backend/main.py
```
